Data Strucuture: 
1 primitive
2 non primitive

Linear Data Structure:  -data elements connected to each other
                        -arranged in sequential manner
                        -Sequential arrangement of the data
                        -each element connect to each other
Ex: LinkedList, Stack, Queue, Array

LinkedList: -chain of pair of data and reference
            -first node: head, last: tail
            -Types: Single-, Double=, Circular@
            -Search:O(n), Insert:O(1), Delete:O(1)

Stack:  -LIFO: Last In First Out
        -A Box
        -push(): insert an element 
        -pop(): remove an element
        -peek(): display top of the elements
        -isempty()
        -isfull()
        -top==-1: stack empty
        -top==0: only one element
        -top==N-1: Stack full
        -top==N: Stack Overflow

Queue:  -FIFO: First In First Out
        -Enqueue(): push element
        -Dequeue(): pop element
        -IsEmpty(), IsFull(), peek()
        -A both side open box
        -two pointers front and rear keep track of elements
        -from only front: Dequeue()

Array: -Traversal, Insertion, Deletion, Search, Update

Non-Linear Data Structure: -Trees, Graph
                           -

Trees:  -Height and Depth of the Trees
        -Root is the first node of Trees
        -nodes are defined as Parant, Child, Internal and leaf
        -Degree of the node= no. of childs of the node
        -Types: Binary, Binary Search, AVL, Red-Black, N-ary 
        -Traversals: DFS:Depth first Search, BFS: Breath First Search
        -Types of DFS Traversal: Preorder, Inorder, Postorder
        -Ex: Pre: 30,20,10,15,25,23,39,35,42
        -In: 10,15,20,23,25,30,35,39,42
        -Post: 15,10,23,25,20,35,42,39,30
